About me

Welcome, I'm Rony Hanna and I'm a huge computer graphics and game programming enthusiast with professional industry experience. I'm currently a full time Software Engineer at MyTona games studio in New Zealand working on an unannounced project. I earned a bachelor's degree in Software Engineering (Game Programming) from Media Design School in Auckland New Zealand. You can view a few of my projects here. Feel free to get in touch, I'd be happy to chat about job opportunities and projects I've worked on.

Projects

An educational AR/VR application made in Unity for iOS and Android. Players create their own virtual worlds, import 3D models from Tinkercad and Google Poly to overlay real world robots, and code their own robot using Python to complete challenges.

Game Jams
